
# Logistic Chase

- This project is a monthly project i made just after learning basic classification and regression models with sklearn and basics of python games with pygame.
- It is going to be a 2 player tag game controlled with keyboard and a human playes against an AI bot




## goals
- To make a playable Game.
- To make this enjoyable anywhere without further training of the model.
- To add graphics to the game along with music and sounds.
## Roadmap

- Build a basic Game structure
- gather the raw data
- refine it up
- train and fine tune model
- export the data
